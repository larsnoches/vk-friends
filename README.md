# VK friends app

Frontend: JS, React  
Backend: Python Django

Debug:
1. `yarn install`
2. `yarn dev`
3. `make setup/dev` (inside activated virtual environment)
4. `make start/dev` (inside activated virtual environment)

Build:
1. `yarn install`
2. `yarn build`
3. `make setup/dev` (inside activated virtual environment)
4. `make refresh/prod` (inside activated virtual environment)
